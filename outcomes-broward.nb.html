Calculating the 10-quantiles for all 3 Arnold PSA scores and all 3 COMPAS scores. Note that for the Arnold NVCA, we use the raw score so as to obtain a quantile; the NVCA is the only one out of the Arnold Scores which has a binary yes/no flag.

scores_quantiles = data.frame( 
  quant_id=seq.int(1,11),
  arnold_nvca_raw = quantile(scores_outcomes$arnold_nvca_raw, probs = seq(0,1,.1),na.rm = T, names = F),
  arnold_nca = quantile(scores_outcomes$arnold_nca, probs = seq(0,1,.1),na.rm = T, names = F),
  arnold_fta = quantile(scores_outcomes$arnold_fta, probs = seq(0,1,.1),na.rm = T, names = F),
  compas_viol = quantile(scores_outcomes$`Risk of Violence_decile_score`,
                       probs = seq(0,1,.1),na.rm = T, names = F),
  compas_recid = quantile(scores_outcomes$`Risk of Recidivism_decile_score`,
                       probs = seq(0,1,.1),na.rm = T, names = F),
  compas_fta = quantile(scores_outcomes$`Risk of Failure to Appear_decile_score`,
                       probs = seq(0,1,.1),na.rm = T, names = F)
  )
